一名拥有 20+ 年 IT 从业经验的工程师，专注于制造业信息化建设、数字化转型和流程优化。

## 🚀 本地部署

```bash
# 克隆仓库
git clone git@github.com:ayeah/ayeah.git
cd ayeah

# 确保使用 Hugo Extended 版本（v0.161.1+）
hugo version

# 本地预览
hugo server -D --baseURL http://localhost:1313

# 构建静态文件
hugo --quiet
```
